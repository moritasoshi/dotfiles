# dotfiles

## Set Up
```sh
$ git clone git@github.com:morita2021/dotfiles.git ~/src
$ cd ~/src/dotfiles && ./install.sh
```

- Optional App
  - Alfred 4
  - Firefox
  - Google Chrome
  - IntelliJ IDEA
  - iTerm
    - sync settings in ~/src/dotfiles/etc/homebrew
  - OmniFocus
  - Perculia
  - Spectacle
    - setting in ~/src/dotfiles/etc/spectacle
  - Visual Studio Code - Insiders
